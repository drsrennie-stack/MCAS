Three operating principles

The rules every generator inherits without question.

01

Inherit, never redefine

Tokens, type, logo, components live here. Generators reference them. No one redefines a color, font size, or hover behavior in a downstream file.

02

Iframe-ready by default

Every generator output ships with the postMessage height-sender and target=_top on internal links. Kajabi, Canvas, anywhere, no layout breakage.

03

WCAG 2.2 AA is the floor

Semantic HTML, real labels, visible focus, color is never the only signal, prefers-reduced-motion honored. Pass the bar before you ship.

Usage rules

What every generator must do to inherit cleanly.

  1. Copy this header lockup verbatim

    Change only the logo-title second word, the logo-sub line, and the hero headline. Leave the structure and class names intact.

  2. Inherit the token block

    Use the swatch palette and Plus Jakarta type stack. Do not introduce a new color, a second typeface, or any italic treatment.

  3. target=_top on internal links

    External links use target=_blank rel=noopener. Internal navigation breaks out of any embedding iframe.

  4. Bake in the iframe height-sender

    PostMessage with a per-file id, ResizeObserver on body, load and resize listeners. Sits before the closing body tag.

  5. WCAG 2.2 AA floor

    Semantic landmarks, single h1, descending hierarchy, real labels, visible focus, prefers-reduced-motion honored, no color-only signals.

  6. No em dashes, no italics

    Em dashes get replaced by commas, periods, parentheses, or rewrites. Italics are never used.

  7. Byline rule

    Student-facing materials sign off as "Dr. Sharilyn Rennie" with no credential suffix. Never ", ND" or ", MD".

  8. No student PII in persistent files

    Names, IDs, emails, and grades are session-only. Nothing student-identifying enters memory, scheduled task logs, or shared cloud folders.

  9. Ship compliance-notes.md per project

    A project is not complete until its compliance-notes file exists. Contrast pairs, keyboard flow, screen reader notes, known limits.
