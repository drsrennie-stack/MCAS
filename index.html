Usage rules for generators

  • Copy the MCAS header block verbatim. Change only H1 and subhead.
  • Inherit the :root token block. Do not redefine colors.
  • Every internal link uses target="_top". External links use target="_blank" rel="noopener".
  • Bake the iframe height-sender script before the closing body tag.
  • WCAG 2.2 AA floor. Visible focus, semantic HTML, real labels.
  • Never em dashes. Use commas, periods, or rewrite.
  • Student-facing byline: "Dr. Sharilyn Rennie" with no credential suffix.
  • Never store student PII in generator output that persists across sessions.
  • Ship a compliance-notes.md alongside every project.
